Case Study

Garbage Management - Professional

2021

Year

HUGGII (Entry Test)

Client

About Project

This project was an entry test for an internship, and I had to develop an application to manage garbage with specifications previously provided. Technologies were imposed so I had to choose between PHP & Symfony. I headed to symfony because I wanted to practice it for some time and it was the perfect opportunity to learn it.

Process

Gathering Requirement

The requirements were that user can create a garbage, while garbage is created it also create a PDF file that can be previewed/downloaded, display the garbage list on a page, garbages can be ordered by submit date, garbages can be ordered by treatment request date (Ajax).

The Problem

This project was the first I did in Symfony, I had to learn new practices and understand them as well as possible in order to apply them correctly and retain them.
